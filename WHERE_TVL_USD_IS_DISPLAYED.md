## User Journey

### Scenario 1: Viewing Overall Protocol TVL
1. User navigates to `/vaults`
2. Sees stats ticker at top
3. Reads "Total TVL: $150,234.56"
4. **This is the aggregated USD value** across all vaults

### Scenario 2: Viewing Individual Vault
1. User clicks on "SEI Hypergrowth Vault"
2. Navigates to `/vault?address=0x123...`
3. Sees vault header with metrics
4. Reads "1234.5678 SEI" in TVL metric
5. **This is the native token amount** (not USD)

### Scenario 3: Comparing Multiple Vaults
1. User on `/vaults` page
2. Sees total TVL in USD at top: "$150K"
3. Scrolls down to vault cards
4. Each card shows native TVL:
   - SEI Vault: "1000 SEI"
   - USDC Vault: "500 USDC"
   - BTC Vault: "0.5 BTC"
