# Demo Mode

Experience Yield Delta risk-free with our comprehensive demo mode that simulates all blockchain interactions while providing realistic user experiences.

## üé≠ Overview

Demo mode allows you to test all platform features without spending real cryptocurrency or interacting with the blockchain. Perfect for:

- **Demo Videos** - Record professional demonstrations
- **User Training** - Onboard new users safely  
- **Development** - Test features during development
- **Strategy Testing** - Experiment with different approaches

## üöÄ Quick Setup

Enable demo mode with a single environment variable:

```bash
# Add to .env.local
NEXT_PUBLIC_DEMO_MODE=true

# Restart your development server
bun dev
```

## üîç Troubleshooting

### Demo Mode Not Working
```bash
# Check environment variable
echo $NEXT_PUBLIC_DEMO_MODE

# Verify in browser console
console.log(process.env.NEXT_PUBLIC_DEMO_MODE)

# Restart development server
bun dev
```

### Transactions Not Simulating
- Verify demo mode indicators are visible
- Check browser console for errors
- Ensure latest code is deployed
- Try hard refresh (Ctrl+F5)

### Performance Issues
- Demo mode should be as fast as real mode
- Check for console errors or network requests
- Verify setTimeout delays aren't too long
- Consider reducing animation complexity
